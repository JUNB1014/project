# 淡水景物辨識導覽系統 
結合 **Android** 與 **TensorFlow Lite** 的影像辨識導覽 APP。
專為淡水遊客設計，提供多國語言的景點導覽服務。

## 🛠️ 使用技術 (Tech Stack)
*   **語言**: Java
*   **AI 模型**: TensorFlow Lite (CNN模型)
*   **開發環境**: Android Studio
*   **功能**:
    *   影像辨識 
    *   多國語言切換 (中/英/日/韓)
